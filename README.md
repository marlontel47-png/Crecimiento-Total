# Crecimiento Total

es un videojuego creado en Python utilizando la librería Pygame.  
El objetivo del juego es controlar una esfera que se desplaza por la pantalla evitando chocar con esferas más grandes, mientras devora esferas más pequeñas para aumentar de tamaño.  
Si el jugador consume suficientes esferas y alcanza el tamaño máximo, gana la partida. Si toca una esfera más grande, pierde.

# Controles

W o Flecha ↑ | Mover hacia arriba
S o Flecha ↓ | Mover hacia abajo
A o Flecha ← | Mover hacia la izquierda
D o Flecha → | Mover hacia la derecha
ENTER | Iniciar juego desde el menú
R | Reiniciar la partida después de perder o ganar
ESC | Salir del juego

# Requisitos del Sistema

Para ejecutar correctamente el juego es necesario contar con:

**Python 3.12**
(Versiones posteriores pueden generar incompatibilidades con Pygame.)
Librería **Pygame** instalada

**Instalar Pygame:**

```bash
pip install pygame
```
